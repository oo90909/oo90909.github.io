hardhat必看.md

hardhat:

chai 的断言库
ether.js看hardhat中chai那一部分
chai

solidity看白皮书
EVM看登链社区


父亲合约能不能像这样被部署，让我以后能够用ether调用被重写的方法？

可以调用未被重写的方法

但即使部署了，也不能调用被重写的方法，参考remix中就知道了，在子合约中肯定只能调用未被重写的方法喽

想要调用被重写的方法肯定只能部署父亲合约了

所以只有私有库需要这样子做

创建hardhat的项目：
1.在hardhat官网上面找到初始化步骤(
记得跟着下载
npm install –save-dev “hardhat@^2.19.1” “@nomicfoundation/hardhat-toolbox@^4.0.0”
)
2.npm install @openzeppelin/contracts 下载openzepplin的包

npx hardhat compile：编译合约（全部）
npx hardhat run script/xxx.js：运行JavaScript代码

如何在成型的项目中添加hardhat

下载： npm install –save-dev hardhat
初始化：npx hardhat init
