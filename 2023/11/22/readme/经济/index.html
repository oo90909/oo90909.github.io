通俗讲解几种经济模型

买方：货币 或 信用
来交换卖方的
商品 服务 金融资产

交易是经济的构成，要理解经济，要先明白，经济是所有市场内的全部交易构成（多方的）

央行控制着经济中的货币和交易数量，通过利率和发行货币来实现这种控制

央行和借款人构成买方和卖方

利率高时，借贷减少，利率低时，借贷便宜

一个人的收入必定是另一个人的支出，因此一个人的生产力决定了他的长期收入，但由于这种生产力不会短期内剧烈变动，因此这不是经济浮动的主要因素，主要因素是信贷。

设想一个只有生产力而没有信贷的世界：

产生了这样的曲线


这张图更形象

在利率低时，人们借钱变得容易，而一个人的支出是另一个人的收入，因此人们的整体货币量会上涨，此时由于货币量和信贷的量上涨，而生产力的提升跟不上货币上涨，生产力出的商品增加不多，便会出现通货膨胀，

而人们通常以为经济曲线上升会一直持续下去，谁曾想，这只是一个阶段，而人们并不知道，他们只关注当下，反而会在此刻借很多钱，但由于周期不长，人们也有能力偿还，所以信誉不会减少

但是，一般人们都是想贷款提前消费，而不去还钱，随着人们所贷金额越来越多，总体社会的债务就会增加，此刻降息也没有用了
